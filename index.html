linix 授予普通root权限

发表于 2018-11-29 | 分类于 linux | | 阅读次数

修改 /etc/sudoers 文件，找到下面一行，在root下面添加一行，如下所示：

Allow root to run any commands anywhere

root ALL=(ALL) ALL
tommy ALL=(ALL) ALL

修改完毕，现在可以用tommy帐号登录，然后用命令 su - ，即可获得root权限进行操作。

git常用命令

发表于 2018-03-27 | 分类于 linux | | 阅读次数

git branch 查看分支
git branch -a 查看所有分支
git checkout 文件 还原指定文件名
git checkout 分支 切换分支
git pull 拉去最新分支
git add 文件夹/. 添加文件到git
git commit -m ‘注释’ 提交git到版本管理
git push 推送版本到远程管理

— git强制更新，覆盖本地文件
git fetch –all
git reset –hard origin/master
git pull

— 查看远程主机的网址
git remote -v
origin git@github.com:jquery/jquery.git (fetch)
origin git@github.com:jquery/jquery.git (push)

— 查看本地分支和远端分支的映射情况
git branch -vv

— 检出新的本地分支，并建立同远程分支的追踪关系
git checkout -b test origin/release

— 检出一个新的分支
git checkout -b newbranch

— git建立远程分支与本地分支追踪关系
git branch –set-upstream-to origin/release

— 合并分支，将dev分支的代码合并至当前分支
git checkout release
git merge dev
git push //将release分支代码推送至远程分支

– 回滚重置到git某次提交
git reset –hard 693343432adfffefefefewaef

docker常用命令

发表于 2018-03-27 | 分类于 docker | | 阅读次数

docker命令
– 进入一个已经正在运行的容器
$ sudo docker ps
$ sudo docker exec -it 775c7c9ee1e1 /bin/bash

docker ps //列出正在运行的容器
docker ps -a //列出所有容器
docker images //查看所有镜像

docker run -p 80:80 –name test -v c:/wampserver/www:/var/www -it 镜像名or镜像id /bin/bash

docker run ：创建一个新的容器并运行一个命令，常用参数:
-i: 以交互模式运行容器，通常与 -t 同时使用；
-t: 为容器重新分配一个伪输入终端，通常与 -i 同时使用
–name=”nginx-lb”: 为容器指定一个名称；
-P 将容器的80端口映射到主机随机端口。
-p 80:80 将容器的80端口映射到主机的80端口
-v 映射主机目录和容器目录

docker cp :用于容器与主机之间的数据拷贝
docker cp /www/runoob 96f7f14e99ab:/www/
