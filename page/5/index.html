linux centos 64位安装mysql步骤

发表于 2017-04-10 | 分类于 linux | | 阅读次数

1、去官网下载
mysql-5.6.35-linux-glibc2.5-x86_64.tar.gz


2、将下载的安装包移动到/usr/local/下,解压

3、复制解压后的mysql目录到系统本地软件目录

4、添加系统mysql组和mysql用户

5、进入安装mysql软件目录，修改目录拥有者为mysql用户

6、安装数据库

7、修改当前目录拥有者为root用户

8、修改当前data目录拥有者为mysql用户,到此数据库安装完毕

9、添加开机启动和启动mysql服务

10、修改mysql的root用户密码，root初始密码为空

11、把mysql客户端放到默认路径。使用软链接过去，便于系统安装多个版本mysql

可能出现的问题

错误信息：ERROR 2002 (HY000): Can’t connect to local MySQL server through socket ‘/tmp/mysql.sock’ (2)
解决方法：打开/etc/my.cnf,看看里面配置的socket位置是什么目录。“socket=/var/lib/mysql/mysql.sock”
路径和“/tmp/mysql.sock”不一致。建立一个软连接：ln -s /var/lib/mysql/mysql.sock /tmp/mysql.sock
之后就可以创建数据库用户，然后使用数据库了。

linux查看开放了哪些端口及如何开启端口

发表于 2017-03-30 | 分类于 linux | | 阅读次数

命令一
netstat -a 查看所有服务端口
命令二
nmap +ip地址
yum install nmap 安装nmap软件包
linux开放服务端口的设置
/etc/sysconfig/iptables
vi iptables
加入如下内容
-A INPUT -p tcp -m tcp –dport 6001 -j ACCEPT
然后保存退出，重启服务
/etc/init.d/iptables restarts
查看端口是否开放
/sbin/iptables -L -n
