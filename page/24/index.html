ionic 项目集成crosswalk，打包apk

发表于 2016-11-28 | 分类于 跨平台移动开发 | | 阅读次数

说明: 这里仅仅是针对android平台，ios的性能没有问题。无需添加crosswalk
1，加入Crosswalk 插件
cordvoa plugin add cordova-plugin-crosswalk-webview
2，获取签名 Android.keystore
可以通过eclipse的androidTools获取，获取通过keytools命令获取。详细的获取方式这里不做介绍
3，设置签名的信息
找到 ../platforms/android/build.gradle
按个人需求修改成如下信息

在../platforms/android 目录下新建文件 release-signing.properties
内容如下：
storeFile=android.keystore
storeType=jks
keyAlias=android.keystore
keyPassword=7788919
storePassword=7788919
4，发布编译版本apk
cordova build android –release
5，发布apk
在 ../platforms/android/build/outputs/apk目录下可以看到生成的文件，将android-armv7-release.apk 发布出去即可

————————–可能会出现的问题—————————-

解决方案：重新打包签名 android.keystore

解决方案：
打开Android SDK Manager
确保Android Suppoer Repository 和 Google Repository 已经安装
