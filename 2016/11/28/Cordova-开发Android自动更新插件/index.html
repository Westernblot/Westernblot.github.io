Cordova 开发Android自动更新插件

发表于 2016-11-28 | 分类于 跨平台移动开发 | | 阅读次数

1.//创建自动更新插件
plugman create –name AutoUpdatePlugin –plugin_id me-plugin-autoUpdatePlugin –plugin_version 0.0.1
2.文件夹下关键资源
E:\WWW\plugin\AutoUpdatePlugin\src\android\WsAutoUpdate.java

E:\WWW\plugin\AutoUpdatePlugin\www\AutoUpdatePlugin.js

E:\WWW\plugin\AutoUpdatePlugin\plugin.xml

————使用插件———–
1.安装cordova 插件
cordova plugin add E:\WWW\plugin\AutoUpdatePlugin
2.删除插件
cordova plugin remove me-plugin-autoUpdatePlugin
3.插件调用

//调用cordova插件检测更新
cordova.plugins.AutoUpdatePlugin.update(null, null, serverVersion, serverUrl, serverDescription);
4.说明，以上数据信息写死了，实际可从服务器上获取。
