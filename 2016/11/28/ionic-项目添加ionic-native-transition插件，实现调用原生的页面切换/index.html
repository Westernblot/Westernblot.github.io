ionic 项目添加ionic-native-transition插件，实现调用原生的页面切换

发表于 2016-11-28 | 分类于 跨平台移动开发 | | 阅读次数

1，ionic-native-transitions github 地址 ：
https://github.com/shprink/ionic-native-transitions
2，添加ionic-native-transition插件
cordova plugin add https://github.com/Telerik-Verified-Plugins/NativePageTransitions
注意： ： Android 如果使用 Crosswalk 并且 Crosswalk 版本 > 1.3 请 把下面的一句话加
载 config.xml 里面
3，nic-native-transitions 结合 ionic 使用

  1. 首先需要 安装 ionic-native-transitions 插件
  2. 在我们的项目中引入 ionic-native-transitions.js 这个 js 在 在 github 上面下载的包里面 ， 看 教程 操作

  3. ionic-native-transitions 依赖注入到我们的 module 里面

  4. $ionicNativeTransitionsProvider 配置 ionic-native-transitions 属性

5,如此可以实现调用原生的页面切换效果
———————-需要注意的问题—————–
当自定义ng-click 路由返回时，$ionicHistory.goBack() 不会调用原生页面切换。
此时可用 $rootScope.$ionicGoBack(); 代替以解决这个问题
应用场景：如双击退出app

标签：

# ionic # 原生页面切换
java 调用 webservice 接口
ionic 项目集成crosswalk，打包apk
